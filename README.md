# yt-list

A small React + Vite demo app that fetches YouTube video data and displays it in a clean, dark YouTube-style interface.

## Features

- Fetches video metadata from a public YouTube videos API
- Displays thumbnails, titles, channel names, and view counts
- Uses a dark, polished YouTube-inspired UI in `src/App.css`
- Responsive video cards with hover emphasis

## Files of interest

- `src/App.jsx` — app logic and video card rendering
- `src/App.css` — YouTube aesthetic styling for the page

## Run locally

```bash
npm install
npm run dev
```

Open the local Vite URL shown in the terminal and view the YouTube-style list.
